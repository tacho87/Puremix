# PureMix Social Media Images

This directory contains images for social media sharing and SEO optimization.

## Required Images:

### OG Images (1200x630px)
- `og-image.jpg` - Main Open Graph image for the landing page
- `og-documentation.jpg` - Open Graph image for the documentation page

### Favicon Images
- `favicon.svg` - SVG favicon for modern browsers
- `favicon-32x32.png` - PNG favicon for browsers
- `favicon-16x16.png` - PNG favicon for browsers
- `apple-touch-icon.png` - Apple touch icon (180x180px)
- `logo.png` - Logo for structured data

### Social Media Icons
- Social sharing buttons use inline SVG icons
- Additional custom icons can be added here

## Creating OG Images:

1. **Recommended dimensions**: 1200x630px for Facebook/LinkedIn
2. **Twitter Card**: 1200x600px for Twitter
3. **Safe zones**: Keep important content within 100px margins
4. **Colors**: Use PureMix brand colors (#6366f1, #8b5cf6, #06b6d4)
5. **Text**: Include PureMix name and value proposition
6. **Logo**: Can include PureMix logo or Python + JavaScript icons
